This directory contents the following files and folders:

	* INPUT_500Gs_30KeV.txt and cst_x_flatBeam.txt – input files for flat beam to 
	  demonstrate Brillouin flow (case 1);
	  
	* INPUT_1000Gs_100KeV.txt and cst_x_divergentBeam.txt – input files for divergent 
	  beam to demonstrate Brillouin flow (case 2);
	  
	* E30keV_I100mA_B500Gs_emittance0.jpg – Hellweg2D results for case 1;
	
	* E100keV_I1mA_B51000Gs_smallEmittance.jpg – Hellweg2D results for case 2;
	
	* folder "edocs":
	
		* radiusDistriburion_1.jpg – radial distribution of initial particles in the files 
		  cst_x*.txt (Cases 1 and 2);
	  
		* angleDistribution_1.jpg  – angle distribution of initial particles in the file 
	      cst_x_divergentBeam.txt (Case 2);
	  
		* verificationOfGilmour.pdf – note with checking of Gilmour’s formulas for Brillouin flow;
	
		* aParm_1mA.jpg, aParm_10mA.jpg, aParm_100mA.jpg, aParm_1000mA.jpg, aParm_10A.jpg, 
	      aParm_100A.jpg, aParm_1000A.jpg – dependence of Gilmour’s parameter “a” for different 
	      beam current and magnetic field;
	  
		* BrillouinFlow.docx and BrillouinFlow.pdf – identical files with Hellweg2D results for 
	      Brillouin flow.
